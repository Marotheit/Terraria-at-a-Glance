<h1>Notes<div id="tooltip"><img src="https://raw.githubusercontent.com/Marotheit/Terraria-at-a-Glance/master/images/tooltip.png"><span id="tooltip_text">The Throwing build is not viable in Hardmode. After that point, this guide will be similar (if not identical) to the Ranged build.</span></div></h1>
	Your first priority should be acquiring <a href="https://terraria.gamepedia.com/Cactus_armor">Cactus armor</a> (or <a href="https://terraria.gamepedia.com/Pumpkin_armor">Pumpkin armor</a> if time allows) and a weapon. Remember, just because you're a throwing item user doesn't mean the importance of a solid weapon can be understated this early in the game. If you come across an <a href="https://terraria.gamepedia.com/Enchanted_Boomerang">Enchanted Boomerang</a> or feel the need to create a <a href="https://terraria.gamepedia.com/Wooden_Bow">Wooden Bow</a>, there's no problem in doing so, since this early in the game you won't be using damage specific armor sets. That being said, throwing falls off <b><i>hard</i></b> the farther you progress into the game, so live it up while you can.<br/><br/>As you approach your first boss, <a href="https://terraria.gamepedia.com/King_Slime">King Slime</a>, you'll want to begin moving towards the Pre-Skeletron build (even if you haven't quite defeated a boss yet) as many of the items can be acquired before fighting any bosses.
